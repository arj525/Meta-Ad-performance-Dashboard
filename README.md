# Meta Ad Performance Dashboard | Power BI Project
This repository contains an end-to-end Power BI dashboard built to analyze and visualize Meta (Facebook &amp; Instagram) advertising performance using structured campaign and user-level data. The project focuses on transforming raw marketing data into actionable insights for performance optimization and strategic decision-making.

## Project Overview
This Power BI dashboard analyzes ad performance data from **Meta (Facebook & Instagram)** to uncover insights into audience behavior, campaign efficiency, and engagement trends.  
Dataset source: **Kaggle**

The project demonstrates my ability to transform raw data into meaningful insights using Excel and Power BI.

## Tools & Techniques Used
- **Excel** → Data cleaning and preprocessing  
- **Power BI** → Data modeling, DAX measures, and dashboard design  
- **DAX** → Calculated KPIs like CTR, Conversion Rate, and Engagement Rate  
- **Data Storytelling** → Insight-driven visual presentation

---

## Key Insights
- **Facebook** achieved higher overall reach (**216K impressions**, **29.3K engagements**) with strong CTR (**11.76%**) and conversions (**5.21%**)  
- **Instagram** maintained a strong **Engagement Rate (13.6%)** and **CTR (11.86%)** despite lower impressions  
- **Story & Video ads** performed best, driving high engagement and conversions  
- **Female users (≈45%)** and **ages 18–30** were the most engaged audience segments  
- Peak engagement observed during **10–12 AM** and **7–9 PM**
